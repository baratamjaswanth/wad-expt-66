# Student Employee Web Application

## Technologies Used
- Java (Servlets)
- JSP
- JDBC
- MySQL
- Apache Tomcat

## Features
- Employee search by ID
- Student registration form
- Database connectivity using JDBC

## How to Run
1. Import project into Eclipse
2. Add MySQL Connector JAR
3. Start Apache Tomcat
4. Open:
   http://localhost:8080/StudentEmployeeApp/employee.jsp
